# Images

Drop your blog images here. Reference them in posts as `/images/filename.png`.

Suggested initial files (replace with your own):

- `favicon.ico`
- `favicon-16x16.png`
- `favicon-32x32.png`
- `apple-touch-icon.png`
- `avatar.jpg` (used if you enable `profileMode` in `hugo.yaml`)
- Cover images per post, e.g. `welcome-cover.png`

Generate favicons easily at https://realfavicongenerator.net/
